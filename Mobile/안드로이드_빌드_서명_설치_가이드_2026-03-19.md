## 2. 빌드용 PC 준비 조건

안드로이드 APK를 만들 PC에는 아래가 필요합니다.

1. **.NET SDK 8/9**
2. **.NET MAUI Android 워크로드**

4. **Android SDK**

## 3. 권장 설치 순서

### 3-1. JDK 설치
JDK 17 이상을 설치하고 새 PowerShell 창에서 아래 명령이 동작해야 합니다.

```powershell
java -version
keytool -help
```

### 3-2. MAUI Android 워크로드 설치

```powershell
dotnet workload install maui-android
```

설치 확인:

```powershell
dotnet workload list
```

출력에 `maui-android` 가 보여야 합니다.

### 3-3. Android SDK 준비
Visual Studio MAUI 워크로드 또는 Android command-line tools 기준으로 준비합니다.

확인 명령:

```powershell
adb version
```

`adb` 는 실제 기기 배포 시 특히 유용합니다.

## 6. 실제 APK 생성 절차

### 6-1. 기본 빌드 명령

```powershell
cd "D:\거래플랜"
powershell -ExecutionPolicy Bypass -File .\tools\mobile\Build-GeoraePlanAndroidApk.ps1 `
  -SigningConfigPath .\Mobile\GeoraePlan.Mobile.App\android-signing.local.json `
  -VersionName "0.1.0" `
  -VersionCode 1
```
