## 2. 이번에 실제로 해결한 점

이 PC는 처음에는 아래 이유로 APK 빌드가 안 됐습니다.

- `maui-android` 워크로드 없음
- `java`, `keytool`, `adb` PATH 미설정
- Android SDK 미설치
- Release 빌드 시 AOT 단계에서 한글 어셈블리 파일명 때문에 실패

현재는 아래 방식으로 해결했습니다.

- 사용자 권한으로 **로컬 .NET 8 SDK** 설치
- 사용자 권한으로 **maui-android workload** 설치
- 사용자 권한으로 **Android SDK** 설치
- 로컬 JDK 경로 자동 탐지
- Release Android APK 빌드에서
  - `RunAOTCompilation=false`
  - `AndroidEnableProfiledAot=false`
  로 설정해 **한글 파일명으로 인한 AOT 실패 회피**
