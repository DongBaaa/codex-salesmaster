# 거래플랜 전체 메뉴 검증 체크리스트

작성일: 2026-03-27

## 1. 목적
이 문서는 거래플랜의 전체 메뉴와 하위 기능을 기준으로,
배포 전/후에 실제 운영 흐름으로 검증할 수 있도록 만든 표준 체크리스트입니다.

사용 원칙:
- 메뉴명만 보지 말고 실제 사용 흐름으로 확인한다.
- 각 메뉴는 `진입 → 조회 → 입력 → 저장 → 재조회 → 수정 → 후속 기능 → 종료 → 재실행` 순서로 본다.
- 관리자/일반유저/담당지점 제한 계정을 나눠 확인한다.
- 빈 상태, 정상 상태, 오류 상태를 나눠 점검한다.
- 동기화 실패 후 재실행 복구까지 확인한다.
