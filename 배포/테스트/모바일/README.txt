거래플랜 Android 테스트

1. Install-Android-Emulator.cmd 를 실행하면 실행 중인 에뮬레이터가 없을 때 자동으로 가상 디바이스를 띄운 뒤 최신 APK를 설치합니다.
2. 직접 설치하려면 이 폴더에 있는 가장 최신 버전의 거래플랜-안드로이드-v0.2.x-signed.apk 파일을 휴대폰 또는 에뮬레이터에 복사해 설치하세요.
3. 앱 내부 서버 주소는 숨겨져 있으며 https://api.example.invalid 로 고정되어 있습니다.
4. APK가 여러 개 있으면 Install-Android-Emulator.ps1 이 가장 최근 수정된 APK를 자동 선택합니다.
